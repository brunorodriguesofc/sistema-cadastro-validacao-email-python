# Sistema de Cadastro e Validação de Email

Projeto desenvolvido para praticar manipulação de dados, validação e lógica de negócio utilizando Python.

## 🔧 Funcionalidades

- Tratamento de strings (nome e email)
- Padronização de dados
- Extração do primeiro nome
- Validação de email
- Identificação do servidor de email
- Cálculo de lucro
- Cálculo de margem de lucro
- Classificação de performance financeira

## 💡 Conceitos aplicados

- Manipulação de strings (strip, lower, title, find)
- Estruturas condicionais (if, elif, else)
- Operações matemáticas
- Formatação de saída (f-strings)
- Lógica de negócio

## ▶️ Como executar

```bash
python main.py
